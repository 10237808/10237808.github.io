Topic 1

Internal Components

Graphics Processing Unit

It allows the computer to render images, animations and video for a display.

Central Processing Unit

It is where all functions and processes are performed on the computer.

Power Supply Unit

Converts A.C. to D.C. at different voltages and supplies power to each component in the PC.

Random Access Memory

Temporary storage which stores any software currently in use for the CPU.

Motherboard

Allocates power and allows communication to the CPU, RAM and all other computer hardware components.

Hard Drive

Permanent form of storage which allows you keep data such as applications or family photos locally on the computer.

CPU Cooler

Draws heat away from the CPU and other components in the chassis.

File Formats

FLAC (Free Lossless Audio Codec)

Lossless format used mostly for music and sometimes as audio tracks on video files.

MKV (Matroska)

Common container for video and audio which allows unlimited number of video, audio, picture and or subtitle tracks in one file.

PNG (Portable Network Graphics)

Uncompressed format used mostly for photos that are stored locally and are not usually used on websites due to their large file size.

PDF (Portable Document Format

Open format used to present documents allowing for text formatting and images.

GIF (Graphics Interchange Format)

A bitmap image format that allows animations as well as a separate palette of up to 256 colours for each frame.

EXE (Executable)

Used on windows to execute code, mostly used to install and run applications such as Google Chrome.

HTML (Hypertext Markup Language)

Most common file format for webpages and web applications allowing for unified scripts in languages such as JavaScript.
